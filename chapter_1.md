## 타입스크립트 사용법

1. 터미널을 연다.
2. npm init 입력 모든 값을 디폴트로 설정
3. npm i @types/node 입력
4. npm install typescript -g (-g로 설치하면 컴퓨터 전체에서 이용할 수 있도록 설치하는 것)
- 맥 OS 면 앞에 sodo를 입력하기
- 권한 오류가 뜨면 프롬프트를 관리자 권한으로 열기

### 타입스크립트를 확인하기 위한 수단

1. npm install ts-node -g
2. ts-node src/index.ts (코드를 터미널에서 확인할 수 있다)

자바스크립트 코드로 컴파일하기
(한개의 파일일 경우)
- tsc src/타입스크립트 위치

### 타입스크립트 컴파일러 옵션 설정하기

- 얼마나 엄격하게 타입 오류를 검사할건지
- 자바스크립트 코드의 버전은 어떻게 할지
 
- tsc --init (설정 초기화)

타입스크립트 설정
```
{
"compilerOptions": {
  "target": "ES5"
},
  "include": ["src"] // 해당 파일안에 있는 타입스크립트를 컴파일한다.
}
```
